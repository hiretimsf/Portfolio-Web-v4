    ## v3.0.0
    **Released:** November 2025

    Built with Next.js 16 and TailwindCSS. Designed and developed from the ground up, inspired by my own headshot. Features a modern, clean, and responsive design, with a focus on performance and SEO optimization.

    **Tech Stack:** Next.js, TypeScript, Tailwind CSS, Shadcn UI

    ![Portfolio v3](/images/apps/web/portfolio-app-v3.jpg)

  </Step>

  <Step>
    ## v2.0.0
    **Released:** March 2025

    Built with Next.js 15 and TailwindCSS. A modern, fast, fully responsive portfolio website with SEO optimization, MDX blog system, and custom react motion animations.

    **Tech Stack:** Next.js, TypeScript, Tailwind CSS, Shadcn UI, TanStack Query, FumaDocs, Content Collection, MDX, Nuqs, Framer Motion, Resend

    ![Portfolio v3](/images/apps/web/portfolio-app-v2.jpg)

  </Step>

  <Step>
    ## v1.0.0
    **Released:** June 2023

    Built with Next.js 13 and TailwindCSS. This was the first version of my portfolio website using a modern framework.

    **Tech Stack:** Next.js, TypeScript, Tailwind CSS, Shadcn UI

    ![Portfolio v3](/images/apps/web/portfolio-app-v1.jpg)

  </Step>
